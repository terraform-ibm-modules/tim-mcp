# TIM-MCP

Terraform IBM Modules MCP Server - A Model Context Protocol server for discovering and working with Terraform IBM Modules.

## Installation

```bash
# Install via uvx (recommended)
uvx tim-mcp

# Or install via pip
pip install tim-mcp
```

## Usage

See [DEVELOPMENT.md](DEVELOPMENT.md) for detailed development and usage instructions.

## Features

- Search Terraform IBM Modules
- Get module details (inputs, outputs, dependencies)
- Browse module examples and solutions
- Retrieve module source code
- Suggest module dependencies from mappings.hcl
